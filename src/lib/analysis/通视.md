<!-- ## 用法 -->

::: info 用法

 打开组件后，鼠标会处于绘制状态

 **直线**：点击任意位置作为起始点，然后再点击一个位置作为目标点，通视分析组件会显示这两个点之间的可视关系，可视部分显示为绿色，不可视部分显示为红色

 **圆形**：点击任意位置作为圆心，再次点击一个位置确定半径，组件会显示圆心四周当前半径范围内的可视关系，可视部分显示为绿色，不可视部分显示为红色

:::
