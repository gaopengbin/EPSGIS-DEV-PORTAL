# 工具条

## 标签

```
<epsplanet-toolbar></epsplanet-toolbar>
```

## 工具条属性

| 属性        | 说明                         | 类型    | 可选 | 默认           |
| ----------- | ---------------------------- | ------- | ---- | -------------- |
| mode        | 模式（vertical、horizontal） | string  | 是   | vertical       |
| position    | 位置                         | object  | 是   | '{top:0,left:0}' |
| collapsable | 可折叠                       | boolean | 是   | false          |
